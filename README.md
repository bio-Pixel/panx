# Academic Homepage Template

A modern, responsive academic homepage template built with Jekyll.

## Features

- Clean, professional design
- Responsive layout
- Easy to customize
- Supports publications, news, and showcase sections
- Social media integration
- Google Scholar and GitHub integration

## Getting Started

1. Install Jekyll and Bundler:
```bash
gem install jekyll bundler
```

2. Install dependencies:
```bash
bundle install
```

3. Run the local server:
```bash
bundle exec jekyll serve
```

4. Open your browser and visit `http://localhost:4000`

## Customization

Edit the following files to customize your homepage:

- `_config.yml`: Site configuration
- `_data/profile.yml`: Personal information
- `_data/navigation.yml`: Navigation menu
- `_data/display.yml`: Display settings
- `index.html`: Homepage layout
