## Asset Preparation Notes

### Image Guidelines:
1. **File Size**: Keep under 2MB per image for optimal loading
2. **Compression**: Use lossless compression for PNG, balance quality vs size for JPEG
3. **Resolution**: Ensure images are high quality (300dpi preferred)
4. **Color Profile**: Use sRGB web-safe colors
5. **Text**: All text should be readable at 1/3 viewport width

### Naming Convention:
- Use lowercase with underscores
- Include descriptive names
- Example: `vascular_atlas_umap_overview.png`

### Placeholder Organization:
- Create directory: `/assets/images/placeholder/`
- Place placeholder images here for initial setup
- After preparing final assets, replace placeholders

### Implementation Steps:
1. Create placeholder images (simple colored boxes with text)
2. Add showcase section to homepage
3. Test layout with placeholders
4. Prepare final assets according to this checklist
5. Replace placeholder images with final assets
6. Review and refine spacing/alignment
